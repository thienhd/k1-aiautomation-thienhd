# Buổi 01 — Cài đặt Hybrid AI Agents (onboarding-automation)

> Nguồn lab: `hocvien_k1_aiautomation/01-onboarding-automation/lab.md`

## Mục tiêu buổi học

Cài đặt Hybrid AI Agents (Antigravity 2.0 / Codex / Claude Code) → gọi "Nhân sự số" AI → quản lý file Skill (`.md`) → đồng bộ GitHub repository học chung.

## KPI / Sản phẩm đầu ra (theo lab.md)

1. Cài đặt thành công **ít nhất 2/3** ứng dụng Hybrid AI: Antigravity IDE, Codex, Claude Code.
2. Tạo & kích hoạt được "Nhân sự số" AI trên máy, sẵn sàng thực thi câu lệnh.
3. Cài đặt trình xem Markdown và đọc/quản lý thành công file Skill `.md`.
4. Git clone thành công `hocvien_k1_aiautomation.git` về máy local.
